# 市盈率和市净率

### 1、每股收益

每股收益又称每股税后利润或者每股盈利：

```latex
公司税后利润(净利润) / 公司总股本
```

### 2、市盈率（PE）

```latex
t / 每股收益   =   公司市值 / 公司净利润
```

### 3、每股净资产值

每股净资产值是公司股东所持有的每股股票所拥有的权益，是表明股票“含金量”的重要指标。

```la
公司净资产 / 发行总股数的值

公司净资产 = 公司总资产 - 公司负债
```

### 4、市净率（PB）

```lat
当前股票价格 / 每股净资产值   =   公司市值 / 公司净资产
```

### 5、净资产收益率（ROE）

```lat
公司净利润 / 公司净资产

净资产收益率 = 市净率 / 市盈率 （ROE = PB / PE）
```



市盈率和市净率从理论上来说，越小则相对越安全。分别代表了你投资该股票的收益与权益
